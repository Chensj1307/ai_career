## 3. 替代率数据模型

### 3.1 四维数据模型

```
替代率 = f(行业, 职业, 学历, 年份)
```

- **行业维度**：20个一级行业
- **职业维度**：每行业5-10个典型职业
- **学历维度**：初中及以下、高中/中专、大专、本科、硕士、博士
- **时间维度**：2025-2030年（年度预测）

### 3.2 替代率定义

- **替代率 (Replacement Rate)**：该职业被AI完全替代的概率
- **人机协作率 (Collaboration Rate)**：AI作为辅助工具增强人类能力的比例
- **转型压力指数 (Transition Pressure Index)**：综合考量替代率和协作率的职业转型紧迫度

## 4. 数据源清单及优先级

### 4.1 数据源分类

| 优先级 | 数据源类型 | 典型来源 |
|--------|------------|----------|
| P0 | 国际权威报告 | 世界经济论坛、麦肯锡、Brookings |
| P1 | AI企业研究 | OpenAI、Anthropic、Google DeepMind |
| P1 | 政府统计数据 | 国家统计局、人社部 |
| P2 | 招聘平台数据 | 智联招聘、Boss直聘、猎聘 |
| P2 | 学术研究 | 高校AI实验室、研究机构 |
| P3 | 行业报告 | 艾瑞、36氪、易观分析 |

### 4.2 数据采集要求

- **数据时效性**：优先使用2023-2025年发布的报告
- **数据覆盖**：确保覆盖所有20个行业和主要职业
- **数据一致性**：建立跨数据源的校验机制
- **更新频率**：核心数据每年更新，预测数据每半年复核

---

## 5. 数据质量标准

### 5.1 准确性要求

- 替代率数值允许误差范围：±10%
- 预测数据需标注置信区间
- 数据来源需可追溯

### 5.2 完整性要求

- 必填字段：行业代码、行业名称、职业代码、职业名称、替代率（至少1年）
- 选填字段：学历影响系数、人机协作率、转型压力指数

### 5.3 一致性要求

- 行业代码与GB/T 4754-2017一致
- 职业代码与《国家职业分类大典》一致
- 时间格式统一为YYYY年
