## 如何添加截图

1. 在浏览器中访问应用：http://localhost:3000
2. 使用截图工具截图
3. 将截图保存为：
   - screenshots/homepage.png
   - screenshots/assessment.png
4. 更新 manifest.json 中的截图链接

## 推荐截图尺寸

- 宽度：1920px 或以上
- 比例：16:9
- 格式：PNG（推荐）或 JPG
