## 🚀 发布步骤

### 步骤1：创建GitHub仓库

1. 访问 https://github.com/new
2. 仓库名称：`ai-career-compass`
3. 描述：`分析各行各业被AI代替的程度和趋势，提供行业×学历双维度分析`
4. 选择 `Public`（公开）
5. 不添加 `.gitignore` 和 `License`
6. 点击 "Create repository"
7. 按照GitHub的提示上传项目文件

### 步骤2：准备应用截图

1. 在本地运行项目：
   ```bash
   cd ai-career-compass/frontend
   npm install
   npm run dev
   ```

2. 访问 http://localhost:3000
3. 截取以下页面：
   - 首页（行业热力图）
   - 个人评估页
   - 行业详情页

4. 将截图保存到：
   - `screenshots/homepage.png`
   - `screenshots/assessment.png`

5. 推荐截图规格：
   - 宽度：1920px 或以上
   - 比例：16:9
   - 格式：PNG（推荐）

### 步骤 3：更新 manifest.json

如果有截图，更新 manifest.json 中的截图链接：

```json
{
  "screenshots": [
    "https://raw.githubusercontent.com/your-username/ai-career-compass/main/screenshots/homepage.png",
    "https://raw.githubusercontent.com/your-username/ai-career-compass/main/screenshots/assessment.png"
  ]
}
```

### 步骤4：在扣子编程商店提交申请

1. 访问扣子编程商店：https://www.coze.cn/skill-store
2. 点击"提交应用"
3. 填写应用信息：
   - **应用名称**：AI职业罗盘
   - **应用描述**：分析各行各业被AI代替的程度和趋势，提供行业×学历双维度分析
   - **应用分类**：工具 > 数据分析
   - **应用标签**：AI分析、职业规划、数据分析、行业研究、风险评估
   - **GitHub仓库**：https://github.com/your-username/ai-career-compass
   - **演示链接**：https://your-deployment-url.com（如果有部署）

4. 上传应用截图
5. 填写技术信息：
   - **前端框架**：Next.js 14
   - **后端框架**：FastAPI
   - **数据库**：PostgreSQL（可选）
   - **开发语言**：TypeScript + Python

6. 提交审核
